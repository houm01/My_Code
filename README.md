## 内容介绍

### python_project

#### Openconnect-project

- 实现的功能
  - 电脑开机后自动登录 VPN，并获 IP 及服务状态信息后通过钉钉发送
- 用到的知识点
  - configparser 模块，通过 ini 读取配置文件
  - 钉钉 API，可以很方便的实现消息发送

#### Cloud_daily_test

- 实现的功能
  - 分析 html 页面，将数据写入到 mongoDB
- 用到的知识点
  - xpath
  - pymongo 模块
